# SHSignatureGenerator

An object for converting audio data into a signature.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+

```
class SHSignatureGenerator
```
