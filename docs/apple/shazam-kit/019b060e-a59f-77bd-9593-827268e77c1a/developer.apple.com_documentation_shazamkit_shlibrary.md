# SHLibrary

An object that represents a user’s synced Shazam library.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+

```
final class SHLibrary
```
